Introduction

idGenerate

用户id生成规则：redis产生自增id,key为v5:auto:uid，拿到自增id后，根据SequenceNumber::generateNumber($autoUid,$prefix='',$width=9)方法生成一个等宽随机uid

autoid在redis中的示例：

key value 描述
v5:auto:uid 27 自增uid
